01 / PHILOSOPHY

為什麼是「魔法」加上「科學」。

創辦人黃新堯來自台大物理系，也是台大魔術社成員。他相信：當孩子第一秒被「魔術般的驚奇」吸引、第二秒就被引導發現背後的科學原理，學習才會真正發生。這個品牌的識別系統，就是把這份「驚喜 → 探究」的體驗，翻譯成可被一致使用的視覺語言。

MISSION

我們存在的理由

打破「科學很枯燥」的刻板印象，讓大班至國中孩子在驚奇與歡笑中，主動探索世界、學會解決問題。

VISION

我們在去的地方

成為全台最具影響力、家長首選的 STEAM 教育精品品牌；讓「魔法科學」成為兒童科學教育的代名詞。

VALUES

我們堅持的原則

做中玩、玩中學。引發興趣是一切的起點；精緻小班、頂尖師資是體驗的底線。

BRAND VOICE / SLOGAN

精彩孩子的人生，
從培養學習興趣開始。

這是創辦人寫給家長的一句話，也是品牌所有對外溝通的第一錨點。在任何延伸文案、課程介紹、海報視覺中，這句話的精神都不能被稀釋。

02 / LOGO

CLEAR SPACE

安全空間 = X

X 為兔子頭部的高度。Logo 周邊任何方向至少保留 1X 距離，避免其他文字、圖形入侵。

MIN SIZE

04 / TYPOGRAPHY

字體：Noto Sans TC × Inter × JetBrains Mono。

05 / MASCOT

認識燒杯兔 — 抱著燒杯的小科學家。

燒杯兔是品牌的吉祥物與情感入口。他是一隻灰色的兔子，戴著有紅色帽帶的紳士帽，總是抱著大燒杯做實驗。個性「會變魔術的聰明大哥哥/大姊姊」。在所有對孩子的素材中，燒杯兔都應該出現至少一次。

PERSONALITY

燒杯兔的人格

  • 🐰 好奇 — 永遠在問「為什麼？」
  • ✨ 友善 — 不說教，會邀請你一起做
  • 🔬 嚴謹 — 實驗一定要做完
  • 🎩 體面 — 紳士帽是他對科學的尊重
VOICE

燒杯兔怎麼說話

對孩子：「咦？這個會發生什麼事？我們一起來試試看！」

對家長：不直接出聲，由品牌正式語氣替代。

在課程教材封面、社群孩子向貼文、教室指標中放置燒杯兔。讓他自然地出現在角落「陪伴」訊息。

07 / COMPONENTS

UI 組件：按鈕、卡片、標籤。

按鈕統一使用「圓角 + 重描邊 + 偏移陰影」的玩具感結構：靜態 3px 偏移、hover 5px、active 0px，模擬可被按壓的實體。卡片用 20px 圓角、2.5px 描邊，重要卡片掛上編號膠帶。

10 / VOICE

語氣：兩個聲道，一個品牌。

家長頁與孩子頁同一個品牌、同一套色彩，但語氣不同。家長端用「專業、穩重、可信任」的方式說話；孩子端用「驚奇、邀請、可動手」的口吻。

TO PARENTS

對家長

  • ✓ DO
    「由台大物理／化學系背景講師親自帶課，4 週 8 堂課完整實作。」
  • ✓ DO
    「每堂課後提供實作紀錄，讓您看見孩子的學習軌跡。」
  • ✗ DON'T
    「超 fun 的科學營等你來，包你小孩玩到不想回家！」
TO KIDS

對孩子

  • ✓ DO
    「咦？這杯水怎麼不見了？我們一起把它變回來！」
  • ✓ DO
    「燒杯兔今天要教你做一個會跳舞的氣球。準備好了嗎？」
  • ✗ DON'T
    「本課程旨在培養學生對於高分子化學的基礎理解。」
